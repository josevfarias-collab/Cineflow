# 🎬 CineFlow

O **CineFlow** é uma plataforma de catálogo de filmes moderna e responsiva, desenvolvida para proporcionar uma experiência fluida de navegação e visualização de trailers. O projeto simula a interface de grandes serviços de streaming, focando em usabilidade e performance.

---

## 🔗 Link para o Projeto
🚀 **[Clique aqui para acessar o CineFlow ao vivo](https://josevfarias-collab.github.io/Cineflow/)**

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do projeto.
* **CSS3**: Estilização avançada com foco em design moderno, uso de Flexbox e Grid.
* **JavaScript (ES6+)**: Manipulação dinâmica do DOM, sistema de login com SessionStorage, filtros de busca em tempo real e integração com player de vídeo.
* **Google Fonts**: Tipografia otimizada (Bebas Neue e Inter).

---

## 🚀 Funcionalidades

* **Sistema de Login**: Acesso controlado via autenticação simulada.
* **Banner Dinâmico**: Destaques que alternam automaticamente com informações dos filmes.
* **Busca Inteligente**: Filtro de pesquisa que atualiza o catálogo conforme o usuário digita.
* **Player Modal Interno**: Visualização de trailers integrada diretamente na interface. Ao clicar no trailer, o vídeo abre em um modal dentro do próprio site, sem redirecionamentos externos.
* **Organização por Categorias**: Filmes agrupados automaticamente por gênero.
